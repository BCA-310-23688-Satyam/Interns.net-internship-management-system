# Internship Management System

Initial full-stack project setup with independent `client` and `server` workspaces.

## Structure

- `client` - React app scaffolded with Vite
- `server` - Node.js + Express API scaffold

## Run

### Client

```bash
cd client
npm install
npm run dev
```

### Server

```bash
cd server
npm install
npm run dev or,
npm start
```
